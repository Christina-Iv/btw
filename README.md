# Адресная Книга
Тестовое задание "Адресная книга" для участия в инкубаторе Frontend разработчиков от Bitworks Software Company.
#### 1. Установка:
```sh
$ npm install
```
#### 2. Запуск:
```sh
$ npm run start
```
Перейти по адресу `http://localhost:8080/` в браузере.
